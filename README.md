# Python Basics — Part 2

> Многопроцессорная симуляция экзаменационной сессии

## О проекте

Проект представляет собой сложную многопроцессорную симуляцию экзамена с использованием `multiprocessing`. Система моделирует взаимодействие студентов, экзаменаторов и вопросов в реальном времени с визуализацией процесса.

## Архитектура

```
┌─────────────────┐     ┌─────────────────┐     ┌─────────────────┐
│   Экзаменаторы  │◄───►│  Общая очередь  │◄───►│    Студенты     │
│  (процессы)     │     │   (Manager.list)│     │  (в очереди)    │
└─────────────────┘     └─────────────────┘     └─────────────────┘
         │                                               │
         └───────────────────┬───────────────────────────┘
                             ▼
                    ┌─────────────────┐
                    │  Монитор (UI)   │
                    │  (отдельный     │
                    │   процесс)      │
                    └─────────────────┘
```

## Ключевые компоненты

### Классы

- **`Student`** — студент с атрибутами: имя, пол, статус экзамена, время начала/окончания
- **`Examiner`** — экзаменатор с логикой обеда, подсчётом времени работы и заваленных студентов
- **`Question`** — вопрос с разбиением на слова для взвешенного случайного выбора
- **`Exam`** — ядро системы: проведение экзамена с золотым сечением для взвешивания ответов

### Алгоритм взвешивания (золотое сечение)

```python
phi = (1 + sqrt(5)) / 2
# Веса распределяются по принципу золотого сечения
# Для женщин — в обратном порядке
```

## Технологии

- **Python 3** `multiprocessing` — разделяемая память через `Manager`
- **Синхронизация** — `Lock` для безопасного доступа к общим данным
- **Визуализация** — ASCII-таблицы в терминале с обновлением в реальном времени

## Интересные моменты

- **Вероятностная модель** — шанс сдачи зависит от случайного броска + соотношения правильных/неправильных ответов
- **Перерыв на обед** — экзаменаторы уходят на обед через 30 секунд после начала
- **Статистика** — подсчёт лучших студентов, экзаменаторов с минимальным процентом завала
